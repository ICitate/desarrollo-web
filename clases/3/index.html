1. Introduccion a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estandar para describir la estructura de los documentos de la web. HTML consiste de una serie de elementos y atributos que se utilizan para "marcar" los componentes de un documento para estructurarlo de una forma significativa.

Los documentos HTML son basicamente un arbol de nodos, estos nodos pueden ser de elementos HTML o texto. Los elementos HTML le dan la semantica y el formato a los documentos. Permiten crear parrafos, listas, tablas, imagenes y formularios.

Un lenguaje de marcado (markup language) es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen informacion adicional acerca de la estructura del texto o su presentacion.

No confundir un lenguaje de marcado con un lenguaje de programacion. Los lenguajes de marcado no tienen funciones aritmeticas, variables ni logica.

2. Elementos

HTML consiste de una serie de elementos que se utilizan para encapsular diferentes partes del contenido para que se vean o funcione de cierta forma. Estos elementos se escriben dentro de tags (etiquetas) entre los simbolos < y >

Para escribir un titulo principal (de nivel 1), se utiliza el tag <h1> . El titulo encerrado dentro de este, es el contenido del elemento. El elemento es la combinacion del tag junto con el contenido. 

<h1>Este es el contenido</h1>

El contenido se escribe entre el tag de apertura y cerrado.
