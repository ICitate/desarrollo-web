1. Introduccion a HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estandar para describir la estructura de los documentos de la web. HTML consiste de una serie de elementos y atributos que se utilizan para "marcar" los componentes de un documento para estructurarlo de una forma significativa.

Los documentos HTML son basicamente un arbol de nodos, estos nodos pueden ser de elementos HTML o texto. Los elementos HTML le dan la semantica y el formato a los documentos. Permiten crear parrafos, listas, tablas, imagenes y formularios.

Un lenguaje de marcado (markup language) es una forma de codificar un documento que, junto con el texto, incorpora etiquetas o marcas que contienen informacion adicional acerca de la estructura del texto o su presentacion.

No confundir un lenguaje de marcado con un lenguaje de programacion. Los lenguajes de marcado no tienen funciones aritmeticas, variables ni logica.

2. Elementos

HTML consiste de una serie de elementos que se utilizan para encapsular diferentes partes del contenido para que se vean o funcione de cierta forma. Estos elementos se escriben dentro de tags (etiquetas) entre los simbolos < y >

Para escribir un titulo principal (de nivel 1), se utiliza el tag <h1> . El titulo encerrado dentro de este, es el contenido del elemento. El elemento es la combinacion del tag junto con el contenido. 

<h1>Este es el contenido</h1>

El contenido se escribe entre el tag de apertura y cerrado. El tag de cerrado es igual que el tag de apertura con un / al inicio.

Los elementos pueden tener otros elementos anidados dentro. Por ejemplo: 

<p>
    Este parrafo contiene texto
    <strong><em>Remarcado</em></strong>
</p>
Resultado:

Este parrafo contiene texto Remarcado

En el codigo anterior podemos observar un elemento p que tiene otros elementos anidados (em y strong). Es importante que los elementos esten anidados de forma correcta.

2.1 Tags principales

Algunos de los tags principales son:

2.1.1 <h1>, <h1>, <h3>, <h4>, <h5>, <h6>

Representan los 6 niveles de titulos de seccion. <h1> siendo la seccion mas alta y <h6> la mas baja. 

<h1>
    Soy un encabezado de nivel 1
</h1>
<h2>
    Soy un encabezado de nivel 2
</h2>
<h3>
    Soy un encabezado de nivel 3
</h3>
Resultado:

Soy un encabezado de nivel 1

Soy un encabezado de nivel 2

Soy un encabezado de nivel 3

2.1.2 <p>

Representa un parrafo. Los parrafos se utilizan para visualizar contenido como bloques de texto separados por saltos de linea. Pero tambien, pueden agrupar imagenes y otros elementos. 

<p>
    Este es el primer parrafo del ejemplo.
    Con mucho texto para enfatizar que 
    es un parrafo
</p>
<p>
    Este es el segundo parrafo del ejemplo.
</p>
Resultado:

Este es el primer parrafo del ejemplo. Con mucho texto para enfatizar que es un parrafo

Este es el segundo parrafo del ejemplo.

2.1.3 <a>

Junto con su atributo href, crea un hipervinculo (link) a paginas web, archivos, o cualquier cosa que se pueda representar con una URL. 

<a href="https://unnoba.edu.ar">
    Esto es un link a UNNOBA
</a>
Resultado: Esto es un link a UNNOBA

2.1.4 <img>

Embebe una imagen en el documento. En el atributo src se especifica la direccion de la imagen. 

<img 
    src="https://shorturl.at/fnpK2"
