1. Links

El Elemento HTML <a> junto con su atributo href, crea un enlace a otras páginas, archivos o ubicaciones dentro de la misma página, direcciones de correo, etc.

Posee la siguiente sintaxis: 

<a href="url" target="_self" >
    texto del link
</a>

1.1 Atributo href

1.2 Atributo target

Por defecto, la página o archivo linkeado será mostrado en la ventana actual en la que nos encontramos. Para cambiarlo, utilizaremos el atributo target

Los valores permitidos por este atributo son:

  1. _self: Valor por defecto. Carga la URL en la misma ventana/tab de navegación actual.
  2. _blank: Carga la URL en un nuevo contexto de navegación. Usualmente es una pestaña nueva, pero los usuarios pueden configurar sus navegadores para que en vez de una pestaña, se abra una ventana.
  3. _parent: Carga la URL en el contexto de navegación padre del actual. Si no existe el padre, se comporta igual que el modo _self.
  4. _top: Carga la URL en el contexto en el contexto más alto de navegación (el cual es un ancestro del actual, y no tiene padre). Si no existe el padre, se comporta igual que el modo _self.
